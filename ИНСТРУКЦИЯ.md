# Инструкция по установке и запуску

**ПРОЕКТ СОЗДАН АВТОМАТИЧЕСКИ ДЛЯ ДЕМОНСТРАЦИИ**

---

## 1. Что нужно установить

```bash
# Скачайте проект
git clone https://github.com/MaxGoodnic/time-series-forecasting.git
cd time-series-forecasting

# Создайте виртуальное окружение
python -m venv venv

# Активируйте окружение
# Mac/Linux:
source venv/bin/activate
# Windows:
venv\Scripts\activate

# Установите библиотеки
pip install -r requirements.txt
```

---

## 2. Как запустить TensorFlow модели

```bash
# Переключитесь на основную ветку
git checkout main

# Запустите обучение
python main.py
```

### Что делает эта команда:
- Скачивает данные о погоде
- Обучает LSTM модели
- Показывает графики
- Сохраняет результаты

---

## 3. Как запустить JAX VAE модели

```bash
# Переключитесь на ветку с VAE
git checkout jax-vae

# Запустите пример
python examples/jax_vae_example.py
```

### Что делает эта команда:
- Обучает VAE на временных рядах
- Показывает восстановление данных
- Демонстрирует интерполяцию

---

## 4. Полезные команды

### TensorFlow:
```bash
python main.py --single-only      # Только одинарные прогнозы
python main.py --multi-only       # Только множественные прогнозы
python main.py --help            # Все опции
```

### Проверка установки:
```bash
python --version        # Версия Python
pip list | grep tf    # Проверка TensorFlow
```

---

## 5. Если что-то не работает

### Проблема: "No module named 'tensorflow'"
```bash
# Переустановите TensorFlow
pip install tensorflow==2.13.0
```

### Проблема: "CUDA error"
```bash
# Установите CPU версию TensorFlow
pip install tensorflow-cpu
```

### Проблема: "Memory error"
```bash
# Уменьшите batch size в коде
# В main.py измените: batch_size = 16
```
